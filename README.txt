PolyPrimer 1.0 ( 2018 )
Author: German Telmo Eizaguirre Suarez
Student of Biotechnology and Computer Engineering at the Rovira I Virgili University (Tarragona, Spain)
for the Biodonostia Research Institute (Donostia, Spain).

Script for detecting polymorphisms at primers of a given gene from the human genotype. Uses the Ensembl API’s mySQL features to acces the Ensembl DB.

Programming language: Perl
Operating system: Linux Ubuntu, preferably

This is a nonprofit project, and anyone could use this code freely.

For installation, follow the guideline in the install.text file.
